# Assets Directory

This directory contains the app assets like images, icons, and fonts.

## Required Files:
- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon foreground
- `favicon.png` - Web favicon

## Getting Started:
1. Add your app icon as `icon.png` (1024x1024px)
2. Add your splash screen as `splash.png`
3. Create adaptive icon foreground as `adaptive-icon.png`
4. Add favicon for web as `favicon.png`

For now, you can use placeholder images or create them using any image editor.
